Jquery Scout Traductores

Jquery Scout Traductores, es un simple plugin que te permite agregar de manera simple traducción de claves scout a tu sitio web por medio de Javascript y jQuery.

Forma de Uso

Inserta en el head de tu sitio web las dependencias de jQuery y el Plugin de Scout Traductores, luego de eso ya estas listo a usar el plugin de la siguiente forma:

$('h1').html($.morse('hola mundo', 'encode'));
$('h2').html($.cenit('hola mundo'));
$('h3').html($.murcielago('hola mundo'));
$('h4').html($.romana('hola mundo'));
$('h5').html($.sufamelico('hola mundo'));
